## Linear-Regression
Introducing people to simple linear regression using python

## Installation required.
  - python 2.7+ or python 3.5+
  - matplot library
  - numpy

## How to run
simply open the terminal and run

    python SimpleLinearRegression.py

The presentation is in frensh
